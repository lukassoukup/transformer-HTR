# transformer-HTR

## Paper

[Pay Attention to What You Read:
Non-recurrent Handwritten Text-Line Recognition](https://arxiv.org/pdf/2005.13044.pdf)

## Instalation

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Pretrained Model

Pretrained model can be downloaded from [here](https://drive.google.com/file/d/1kPJpSPsykbm-duwK_tl-hTr2XcrqwqIU/view?usp=sharing)

## Execution

*predict.py* --input_dir --model_file [--output_dir] 
- _input_dir_ - path to directory with input images
- _model_file_ - path to model pretrained model file
- _output_dir_ - output directory
